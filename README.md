# 🚀 Lorenzo Airoldi · Portfolio

[![GitHub Pages](https://img.shields.io/badge/hosted-GitHub%20Pages-326ce5?style=flat&logo=github)](https://lorenzoairoldi-lab.github.io/Portfolio)
[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> **Infrastructure as code today, resilient systems tomorrow.**

Personal cloud engineering portfolio built with vanilla HTML, CSS, and JavaScript. Features an interactive dark-themed UI with particle animations, dynamic filtering, and a fully responsive layout.

## ✨ Features

- **Home** — Hero section with animated particles, typewriter effect, and live stats counter
- **About** — Personal bio with interactive terminal display and skill tags
- **Skills** — Categorized skills with animated progress bars (Cloud & DevOps, Linux & Networking, Development)
- **Certifications** — Dynamic certification cards with category filtering, badge modals, and progress tracking
- **Projects** — Project showcase with tech stack tags and GitHub links
- **Contact** — Contact information and functional contact form
- **Fully responsive** — Optimized for mobile, tablet, and desktop
- **Accessible** — ARIA labels, keyboard navigation, reduced-motion support

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Semantic markup, accessible structure |
| CSS3 | Custom properties, CSS Grid, Flexbox, container queries |
| Vanilla JavaScript | Intersection Observer, Canvas API, dynamic data loading |
| GitHub Pages | Hosting and deployment |
| Google Fonts | Space Grotesk + JetBrains Mono |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/lorenzoairoldi-lab/Portfolio.git

# Serve locally with Python
cd Portfolio
python -m http.server 8080

# Or use any static server
npx serve .
```
